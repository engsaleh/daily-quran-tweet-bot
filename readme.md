
# Daily Quran Tweet Bot | بوت التغريد اليومي للقرآن الكريم

> يقوم هذا البوت بنشر آية قرآنية بشكل يومي على منصة X (تويتر) دون أي تدخل بشري، وبدون تكاليف استضافة.

## ❓ لماذا هذا المشروع؟

نشر آيات القرآن الكريم وتذكير الناس بها هو عمل عظيم، ولكن النشر اليدوي اليومي قد يكون صعباً بسبب الانشغال بالمهام اليومية. هذا المشروع يحل هذه المشكلة من خلال **أتمتة العملية بالكامل**؛ حيث يقوم بجلب آية عشوائية، تنسيقها، ثم نشرها يومياً بشكل مستقل.

---

## 💡 المميزات

* **أتمتة كاملة 100%:** يعمل يومياً في وقت محدد مسبقاً باستخدام `GitHub Actions`.
* **مجاني بالكامل:** لا حاجة لدفع تكاليف سيرفرات أو استضافة سحابية.
* **تجدد مستمر:** يعتمد على [Quran Cloud API](https://alquran.cloud/api) لجلب آيات عشوائية بشكل مستمر وموثوق.
* **سهل التخصيص:** يمكنك بسهولة تعديل الكود لإضافة أذكار، وسوم (Hashtags) معينة، أو حتى ترجمة للآيات.

---

## 🧠 كيف يعمل؟ (مسار العمل)

1. **المؤقت الزمني (Cron Job):** تقوم خوادم GitHub بتشغيل الكود يومياً في تمام الساعة المحددة.
2. **جلب البيانات:** يتصل الكود بـ API القرآن الكريم لجلب آية عشوائية مع اسم السورة ورقم الآية.
3. **التنسيق:** تتم معالجة النص ليظهر بشكل جميل ومريح للقراءة داخل التغريدة.
4. **النشر:** يتصل الكود بواجهة تويتر (X API) وينشر التغريدة مباشرة على حسابك.

## ⚙️ متطلبات التشغيل

قبل البدء، ستحتاج إلى:

* حساب على منصة **GitHub**.
* حساب مطورين على منصة X ([Developer Portal](https://developer.x.com)) على الباقة المجانية (Free Tier).
* مفاتيح X API التالية بصلاحية قراءة وكتابة (**Read and Write**):
* `API_KEY`
* `API_SECRET`
* `ACCESS_TOKEN`
* `ACCESS_TOKEN_SECRET`

## 🚀 طريقة التثبيت والإعداد

### 1. استنساخ المستودع

قم بعمل **Fork** لهذا المستودع إلى حسابك الشخصي، أو قم بنسخ الملفات الموجودة هنا إلى مستودع جديد خاص بك.

### 2. إضافة المفاتيح السرية (Secrets)

لحماية مفاتيحك الخاصة بتويتر من السرقة، يجب عدم وضعها داخل الكود مباشرة. بدلاً من ذلك، اتبع التالي:

* اذهب إلى صفحة المستودع الخاص بك على GitHub واضغط على **Settings**.
* من القائمة الجانبية، اختر **Secrets and variables** ثم **Actions**.
* أضف المفاتيح الأربعة التالية كـ **New repository secret**:
* `TWITTER_API_KEY`
* `TWITTER_API_SECRET`
* `TWITTER_ACCESS_TOKEN`
* `TWITTER_ACCESS_TOKEN_SECRET`



### 3. تفعيل GitHub Actions

* اذهب إلى تبويب **Actions** في أعلى المستودع.
* إذا ظهرت لك رسالة تطلب منك تفعيل بيئة العمل، اضغط على زر **"I understand my workflows, go ahead and enable them"**.
* لتجربة البوت فوراً؛ اختر **Daily Quran Tweet** من القائمة الجانبية اليسرى، ثم اضغط على زر **Run workflow** الموجود على اليمين.

## 🛠️ التخصيص

* **تعديل وقت النشر:** يمكنك تغيير موعد النشر من خلال ملف `.github/workflows/daily_tweet.yml` عن طريق تعديل السطر: `cron: '0 6 * * *'` (الافتراضي هو 6:00 صباحاً بتوقيت UTC).
* **تعديل التنسيق:** يمكنك تغيير شكل التغريدة وإضافة نصوصك الخاصة من خلال التعديل على دالة `fetch_quran_verse()` الموجودة في ملف `script.py`.
